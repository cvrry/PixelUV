# PixelUV

A blender addon that takes a family of objects with named vertex groups and maps them onto a single unique point on a UV Map

Each object is a "level" and each vertex group is a "zone"

Obj Data  ->  UVmap<br />
zones     ->  X axis<br />
levels    ->  Y axis<br />
